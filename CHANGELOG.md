# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Contributing guidelines
- Code of Conduct
- Security policy
- Issue and PR templates
- CI workflow for automated linting
- EditorConfig for consistent code style

### Changed
- Updated package.json with proper npm fields
- Improved .gitignore configuration
- Enhanced README with badges and better structure

## [1.2.0] - 2023

### Added
- Powerful set of tools and functionalities to create interactive Gantt charts
- Support for task dependencies and links
- Customizable columns and scales
- Theme support with dark theme
- Comprehensive documentation

### Features
- Interactive drag-and-drop functionality
- Task progress tracking
- Multiple view scales (day, week, month)
- Tree structure for project hierarchy
- Link types for task dependencies
- Responsive design

## [1.0.0] - 2023

### Added
- Initial release of javascriptgantt
- Basic Gantt chart functionality
- Task management
- Timeline visualization
